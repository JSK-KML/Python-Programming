# Tutorial 0 - Getting Started

Before we write a single line of code, let's set up the tools and identity you will use for the rest of this course. A clean, professional account makes your work easy to recognise and easy to share. Work through each task below in order.


## Task 1 - Create a Gmail Account <Badge type="tip" text="Task" />

Create a **new Gmail account** that uses only your **name** or a **short version of your name**.

**Rules for the email address:**

- Use your name or short name only.
- **No numbers.**
- **No special symbols** except `_` , `-` and `.`

| Good | Avoid |
|------|-------|
| `ali.hassan@gmail.com` | `ali12345@gmail.com` |
| `siti_nurhaliza@gmail.com` | `siti.cute.99@gmail.com` |
| `ahmad-zaki@gmail.com` | `xX_ahmad_Xx@gmail.com` |


## Task 2 - Take a Profile Picture <Badge type="tip" text="Task" />

Take a **picture of yourself** to use as your profile photo.

**Guidelines:**

- It should look **professional**, but **not too formal**. This is *not* a passport photo.
- You may **go outside the class** to find good lighting or a nice background if needed.
- **Look happy**: smile, relax, be yourself.
- Make sure your face is clearly visible.

**Tips:**

- Use natural light (near a window or outdoors) for the clearest photo.
- A plain or simple background keeps the focus on you.
- Frame from the chest up.



## Task 3 - Create a GitHub Account <Badge type="tip" text="Task" />

Create a **GitHub account** at [https://github.com/signup](https://github.com/signup) using the **Gmail address you made in Task 1** as your login.

**Rules for your GitHub username:**

- Choose a username people can **recognise you by**.
- Keep it **simple and not too formal**.

| Good | Avoid |
|------|-------|
| `ali-codes` | `user48823` |
| `sitidev` | `coding_legend` |
| `zaki-builds` | `qwerty_xyz` |



## Task 4 - Add Your Profile Picture <Badge type="tip" text="Task" />

Set the **photo from Task 2** as your profile picture in **both**:

1. Your **Gmail / Google account**
2. Your **GitHub account**

Using the same photo in both places makes your identity consistent and easy to recognise.



## Checklist <Badge type="warning" text="Submit" />

Before you finish, make sure you have completed all of the following:

- New Gmail created (name only, no numbers, no symbols except `_ - .`)
- Professional but friendly profile photo taken
- GitHub account created using the new Gmail
- Recognisable GitHub username chosen
- Profile photo added to **both** Gmail and GitHub
